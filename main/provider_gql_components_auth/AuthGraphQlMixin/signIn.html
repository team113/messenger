Creates a new Session for the MyUser identified by the provided num/login/email/phone (exactly one of four should be specified).

Represents a sign-in action.

The created Session has expiration, which may be prolonged via renewSession if the remember argument is specified (so a RememberedSession is returned).

Authentication

None.

Non-idempotent
