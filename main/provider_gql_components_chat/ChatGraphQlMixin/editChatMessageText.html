Edits ChatMessage's text by the authenticated MyUser.

ChatMessage's text is allowed to be edited within 5 minutes since its creation or if it hasn't been read by any other Chat member yet.

Authentication

Mandatory.

Result

Only the following ChatEvent may be produced on success:

  • EventChatItemTextEdited.

Idempotent

Succeeds as no-op (and returns no ChatEvent) if the given ChatMessage's text is already set to the given value.
