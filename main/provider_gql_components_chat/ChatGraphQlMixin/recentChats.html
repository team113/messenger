Returns non-hidden Chats of the authenticated MyUser ordered descending by the last updating time.

It's allowed to specify both first and last at the same time, provided that after and before are equal. In such case the returned page will include the Chat pointed by the cursor and the requested number of Chats preceding and following it.

If it's desired to receive the Chat pointed by the cursor without querying in both directions, one can specify first or last as 0.

Authentication

Mandatory.

      operationName: 'RecentChats',

      variables: variables.toJson(),
    ),
  );
  return RecentChats$Query.fromJson(result.data!);
}
