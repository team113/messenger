bool isSvg

Indicates whether this file represents an SVG or not.

Implementation

bool get isSvg {
  // Best effort if [mime] is `null`.
  if (mime == null) {
    return extension == 'svg';
  }

  return mime?.subtype == 'svg+xml';
}
