Adds a new phone number for the authenticated MyUser.

Sets the given phone number as an unconfirmed sub-field of a MyUser.phones field and sends to this number SMS with a ConfirmationCode.

Once MyUser successfully uses this ConfirmationCode in a Mutation.confirmUserPhone, the phone number becomes a confirmed one and moves to MyUserPhones.confirmed sub-field unlocking the related capabilities.

MyUser can have maximum one MyUserPhones.unconfirmed number at the same time.

Authentication

Mandatory.

Result

Only the following MyUserEvent may be produced on success:

  • EventUserPhoneAdded.

Idempotent

Succeeds as no-op (and returns no MyUserEvent) if the provided phone already is present in a MyUser.phones field (either in confirmed or unconfirmed sub-field).
