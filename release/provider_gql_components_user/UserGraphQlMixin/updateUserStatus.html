Future<MyUserEventsVersionedMixin?> updateUserStatus(

Updates or resets the MyUser.status field of the authenticated MyUser.

Authentication

Mandatory.

Result

One of the following MyUserEvents may be produced on success:

  • EventUserStatusUpdated (if text argument is specified);
  • EventUserStatusRemoved (if text argument is absent or is null).

Idempotent

Succeeds as no-op (and returns no MyUserEvent) if the authenticated MyUser has the provided text as his status value already.

Implementation
